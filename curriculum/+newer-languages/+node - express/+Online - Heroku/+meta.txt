# These are the basic steps to deploy an Express server to heroku:
1. Login at heroku.com
2. Create app.
3. Install heroku CLI (try using brew)
...
4. For pushing up your app to heroku, you must have
- It must be a git repository, so you can push with:
```
git push heroku master
```
- There must be a package.json at the root because everytime you push, it reinstall npm modules and runs the script start.
```
"scripts": {
    "start": "node server.js"
}
```
