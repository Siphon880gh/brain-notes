# Git repositories with "node_modules/"

Before making your first commit, untrack "node_modules/" by adding it to .gitignore:
```
echo "node_modules/" >> .gitignore
```

If the node_modules folder was already tracked by git, then remove the tracking from cache, add it to .gitignore, and then commit again.
```
git rm -r --cached node_modules
echo "node_modules/" >> .gitignore
git add -A
git commit -m "YOUR_MESSAGE"
```
